====================================

the simulation components from the active site. Move (do NOT delete)

§B ACCEPTANCE
-------------
B1.  Every moved file lives under src/_legacy_v1/. The move preserves
     directory structure under the legacy root for archeology.

B2.  The site still BUILDS. The current routes in src/app/ that import
     the moved components are temporarily edited to render a stub
     placeholder ("v2 under construction") instead of importing legacy
     components. The stub is a single shared component:

       src/components/StubPlaceholder.tsx
         — renders centered text: "this page is being rebuilt in v2.
            see /docs/v2_audit.md for the plan."
         — used by every page.tsx whose original imports were moved.

     This means: src/app/page.tsx, src/app/{about,resume,hivemind,
     purposeless-efficiency,theseus,projects/[slug]}/page.tsx all
     import StubPlaceholder for now. Suites 14-16 replace these stubs
     with real v2 pages.

B3.  Tests that exercise removed components are SKIPPED, not deleted.
     Each test file gets a `describe.skip(...)` wrapper with a comment:
       // SKIPPED: legacy v1 component, see _legacy_v1/. Restored when v2 covers same surface.
     Files affected (search before editing): tests/accordion/*,
     src/stores/coherenceEngineExperience*.test.ts,
     src/stores/theseusExperience.test.ts, e2e/*.spec.ts that hit
     the accordion.

B4.  Imports that broke from the move are surfaced. After the move,
     run `npm run build` and capture every "Cannot find module" or
     TypeScript error. Each error must be addressed by either:
       (a) updating the import to reference _legacy_v1/ (only if the
           caller is itself moved to _legacy_v1/), or
       (b) replacing the caller's usage with StubPlaceholder.
     There must be ZERO unhandled build errors at the end.

B5.  The build output is documented at the bottom of docs/v2_audit.md
     under a new section "## Demolition build state" that lists
     (i) the routes now rendering StubPlaceholder, (ii) any tests
     skipped, (iii) any dependencies that became unused.
