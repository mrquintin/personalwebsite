P03 — MODE 2, REFLECTION LAB
===============================

ROLE: You are implementing MODE 2 of the Theseus Deep experience.
The Reflection Lab makes the Householder reflection — the
mechanism behind Theseus's adversarial challenge and Dialectical
Crucible — legible through a sequence of pre-computed,
source-faithful fixtures. The user sees why naive negation fails,
how the reflection operates on embedding vectors, and what the
nearest real-world sentences to the reflected vectors look like.

Read P01 and P02 first. This prompt assumes the
chassis is in place and OrdersSlice is functional. It augments
the store with ReflectionSlice and adds four (thesis, axis,
precomputed projections, nearest-neighbor corpus sentences)
fixtures.

GROUNDING

All content in this mode is grounded in:

  TH3 §3     — the Embedding Geometry Conjecture (esp. 3.1, 3.2)
  TH3 §5     — Reverse Marxism: ideology flipping by concept-axis
               reflection (all subsections)
  TH3 §5.1   — Why reflection rather than negation
  TH4 §2.2   — geometry.py module (contradiction-geometry detection
               via Householder reflection and difference-vector
               sparsity)
  TH6 §3     — The Dialectical Crucible (Reverse Marxism in
               production form)

The Householder formula (verbatim, TH3 §5):

    Given an embedding vector v and a unit vector â that
    represents a concept axis ("class," "liberty,"
    "efficiency," etc.), the Householder reflection across the
    hyperplane orthogonal to â is

        v' = v − 2 (v · â) â.

    This reflection is an involution: applying it twice
    returns v.

"Why reflection rather than negation" (verbatim, TH3 §5.1, two
paragraphs — both land in sources.ts and in the UI):

    "Simple negation produces nonsense: 'capitalism exploits
    workers' becomes 'capitalism does not exploit workers,'
    which is a denial rather than a constructive counter-
    framework. Reflection preserves the structure of the
    argument while inverting its orientation. Marx's analysis
    of how value flows through the economy — the fact that
    there are systematic flows between classes — is preserved
    under reflection. What changes is the direction of the
    flow: instead of 'value extracted from worker to capitalist
    (exploitation),' the reflected structure describes 'value
    delivered from capitalist to worker (enrichment).' The
    reflected text reads as a systematic pro-capitalist
    argument with the structural rigor of Marx, because it
    inherits that structural rigor under the reflection
    operation."

The research hypothesis (TH3 §5.2, verbatim):

    "The reflected structure IS capitalism's own self-
    justification — the argument that capitalism makes for
    itself, articulated with the same depth and systematicity
    that Marx brought to his critique."

These three excerpts MUST land verbatim in
src/lib/theseus/sources.ts with the keys:

    TH3_SEC_5_HOUSEHOLDER_DEF      // the formula paragraph
    TH3_SEC_5_1_WHY_REFLECTION     // the two paragraphs

C1. Fixture shape

    // All numeric arrays below are PRECOMPUTED. Keep magnitudes
    // faithful to what TH3 §3 reports for real embeddings:
    //    - cos(original, reflected) across a class axis
    //      typically ≈ 0.45-0.65 (not opposite vectors);
    //    - difference-vector sparsity (Hoyer) ≈ 0.7-0.85
    //      concentrated in ~14% of dimensions;
    //    - projection onto â varies 0.3-0.9 depending on how
    //      axis-aligned the sentence is.
    // The illustrative 2D projections below REPRESENT these
    // magnitudes in a 2D slice chosen so the reflection is
    // visually clear. They are NOT raw embeddings.

    export interface Sentence {
      text: string;              // the actual sentence
      projectionOrig: [number, number];    // 2D illustrative
      projectionReflected: [number, number];
      nearestCorpus: {
        // after reflection, the nearest authored sentence from
        // the reference corpus
        text: string;
        source: string;          // e.g., "Smith, Wealth of
                                 // Nations, Bk I Ch 2"
      };
      dotProductWithAxis: number;   // v·â in [-1, 1]; drives the
                                    // reflection magnitude
    }

C6. Authoring discipline

    - The nearestCorpus entries MUST be labeled as authored /
      paraphrased / representative when they are paraphrased
      rather than verbatim quotations from a published work.
      Paraphrase is acceptable for pedagogical clarity; silent
      attribution to a named book of a sentence the author did
      not write is NOT. The P07 copy lint inspects every
      nearestCorpus.text to require either (i) matching a
      citation-verified source in a build-time corpus file, OR
      (ii) a "(paraphrased for illustration)" or
      "(paraphrased)" tail. The four fixtures above mostly
      use paraphrase; this is by design.

    - All numeric values in projectionOrig / projectionReflected
      are ILLUSTRATIVE. They satisfy:
        projectionReflected = (−projectionOrig[0], projectionOrig[1])
      i.e., the axis â is taken to be the unit x-axis in the
      2D illustration, so the Householder reflection's effect
      is to negate the x-component and preserve y.

    - dotProductWithAxis equals |projectionOrig[0]|; it is
      the magnitude of the projection onto the axis.

D4. EmbeddingProjection.tsx — step (ii)

    An SVG canvas (400x300 on STANDARD+; 320x240 on NARROW)
    rendering:
      - A 2D coordinate grid with axes labeled "â (axis)"
        on the x-axis and "orthogonal" on the y-axis.
      - The axis-aligned dashed line along the x-axis (the
        axis of reflection).
      - The hyperplane orthogonal to â (the y-axis itself —
        the line of reflection).
      - Each sentence rendered as a dot at its
        projectionOrig.
      - Next to each dot: the sentence index (1, 2, 3, ...)
        as a small labeled badge.

    A REFLECT / UNREFLECT toggle (D7) drives an animated
    transition of the dots from projectionOrig to
    projectionReflected. During the animation (reducedMotion
    collapses to instant swap):
      - The dot's x-coordinate inverts (animates via
        2*dotProductWithAxis*axis_vector subtraction).
      - A faint arc connects original -> reflected for 400ms.

    Above the SVG, the illustration label:

      "Illustrative projection of precomputed embeddings;
       not live embeddings."

    This label is REQUIRED and checked by the P07 lint. Its
    font size is the same as body copy; it is not hidden or
    de-emphasized.

    Below the SVG:
      - A per-sentence readout: tap a sentence dot and the
        readout shows:
          "Sentence {i}:
             v   = ({x},    {y})
             v·â = {dotProductWithAxis}
             v'  = ({-x},   {y})    (reflection)"
      - A note under the readout:
          "Axis â taken as the x-axis in this illustration.
           Under the Householder reflection, the
           x-component (the axis-aligned component) is
           inverted; the y-component (the orthogonal
           component) is preserved." (This is authored
           explanatory prose.)

    export function typeGuardForThesisAxes(x: unknown): ThesisAxis[] {
      if (!Array.isArray(x)) throw new Error("THESIS_AXES not array");
      if (x.length !== 4) throw new Error("need 4 fixtures");
      const ids = new Set<string>();
      for (const t of x as any[]) {
        for (const k of ["id","label","axisLabel","axisVectorLabel",
                         "thesisTitle","thesisSummary","referenceCorpus",
                         "sourceAnchor"]) {
          if (typeof t[k] !== "string") throw new Error(`${t.id || "?"}.${k}`);
        }
        if (ids.has(t.id)) throw new Error(`dup id ${t.id}`);
        ids.add(t.id);
        if (!Array.isArray(t.sentences) || t.sentences.length < 3) {
          throw new Error(`${t.id}.sentences length`);
        }
        for (const s of t.sentences) {
          for (const k of ["text","authoredNegation"]) {
            if (typeof s[k] !== "string") throw new Error(`${t.id} sentence.${k}`);
          }
          if (!Array.isArray(s.projectionOrig) || s.projectionOrig.length !== 2) {
            throw new Error(`${t.id} projectionOrig`);
          }
          if (!Array.isArray(s.projectionReflected) || s.projectionReflected.length !== 2) {
            throw new Error(`${t.id} projectionReflected`);
          }
          // Illustrative-axis invariant: reflection negates x-component,
          // preserves y-component.
          if (Math.abs(s.projectionReflected[0] - (-s.projectionOrig[0])) > 1e-9) {
            throw new Error(`${t.id} projection reflection invariant`);
          }
          if (Math.abs(s.projectionReflected[1] - s.projectionOrig[1]) > 1e-9) {
            throw new Error(`${t.id} projection orthogonal preservation`);
          }
          if (typeof s.dotProductWithAxis !== "number") {
            throw new Error(`${t.id} dotProductWithAxis`);
          }
          if (Math.abs(s.dotProductWithAxis - Math.abs(s.projectionOrig[0])) > 1e-9) {
            throw new Error(`${t.id} dot-product must equal |projectionOrig[0]|`);
          }
          if (!s.nearestCorpus || typeof s.nearestCorpus.text !== "string"
              || typeof s.nearestCorpus.source !== "string") {
            throw new Error(`${t.id} nearestCorpus shape`);
          }
        }
        if (!t.involutionCheck
            || typeof t.involutionCheck.tolerance !== "number"
            || typeof t.involutionCheck.maxAbsError !== "number") {
          throw new Error(`${t.id} involutionCheck`);
        }
        if (t.involutionCheck.maxAbsError >= t.involutionCheck.tolerance) {
          throw new Error(`${t.id} involution drift exceeds tolerance`);
        }
      }
      return x as ThesisAxis[];
    }
