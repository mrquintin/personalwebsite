       ## Prerequisites
         - Vercel account
         - GitHub repo connected to Vercel project
         - Anthropic Console account (API key)
         - Voyage AI account (API key)
         (or OpenAI if user opts; doc both)

       ## First-time setup
         1. In Vercel dashboard → Storage → Create Postgres database.
            Link to project. (Auto-populates POSTGRES_URL,
            POSTGRES_URL_NON_POOLING.)
         2. In Vercel dashboard → Storage → Create KV (Upstash Redis).
            Link to project. (Auto-populates KV_REST_API_URL,
            KV_REST_API_TOKEN.)
         3. In Vercel project Settings → Environment Variables, add:
              ANTHROPIC_API_KEY
              VOYAGE_API_KEY (or OPENAI_API_KEY)
              EMBEDDER (default "voyage")
              NEXT_PUBLIC_SITE_URL (set to production URL)
              LLM_MODEL (default "claude-opus-4-7")
              LLM_RATE_LIMIT_HOUR / _DAY (defaults from env.ts)
            Set scope to Production + Preview + Development as
            appropriate.
         4. Pull env locally: `vercel link` then `vercel env pull`.
         5. Run db migrations: `npm run db:migrate`.
         6. Seed corpus: drop initial files into corpus/, edit
            corpus/manifest.json, then `npm run ingest`.
         7. First deploy: `vercel --prod` or push to main
            (auto-deploy via GitHub integration).

       ## Subsequent deploys
         - Merge to main → Vercel auto-builds + deploys.
         - Sitemap regenerated by postbuild.
         - PDFs regenerated by build-pdf script (16/P08).

       ## Updating the corpus
         - Add or edit files under corpus/.
         - Update manifest.json with new entries.
         - Run `npm run ingest`.
         - Confirm chunks appear in DB:
             psql $POSTGRES_URL -c "select count(*) from chunks;"
         - Optionally run `npm run evals:llm` to spot-check voice/
           groundedness.

       ## Cost expectations
         - Anthropic API: rough estimate per chat message
           (claude-opus-4-7 is the priciest tier). The doc states a
           ballpark "~$0.02-0.10 per chat message" with
           rate limits keeping the daily ceiling bounded.
         - Voyage embeddings: ~$0.0001 per 1K tokens. A corpus of
           ~500K tokens costs <$0.10 to ingest once.
         - Vercel Postgres: free tier viable for v2-scale corpus.
         - Vercel KV: free tier viable for v2-scale traffic.
         - Total ongoing: bounded by rate limits; ~$5-20/month
           expected at low traffic.

       ## Rollback
         - Vercel keeps previous deployments. Rollback via dashboard.
         - DB migrations are forward-only by default; for a schema
           rollback, manually run a down-migration.

       ## Troubleshooting
         - "Unauthorized" on chat → check ANTHROPIC_API_KEY.
         - "No relevant material" on every query → corpus is empty
           or ingestion failed; re-run with --rebuild.
         - 429s spiking → check KV is connected; raise LLM_RATE_LIMIT_*
           cautiously.
         - PDF resume missing → build-pdf script failed; check
           Puppeteer/Playwright in Vercel deployment logs.
