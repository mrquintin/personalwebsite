PROMPT SET 14 — NEW SHELL AND NAVIGATION
==========================================

CONTEXT
-------
Replace the v1 accordion shell with a conventional Next.js app
shell: a global header with nav links, a global footer, page-level
routes for content, a mobile drawer, page transitions, and a
theme toggle.

Per principle 1 (Pages, not panels): the navigation primitive is
the URL. Per principle 5 (The visitor can always leave): the global
nav is present on every page.
