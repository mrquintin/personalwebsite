THE GLOBAL VISION (read this once, then forget)
-----------------------------------------------
The site is "The Operator's Portfolio." Not a designer's portfolio. A
workstation. The user (Michael) is a thinker — companies, books, knowledge
systems — and the site should signal that immediately by feeling like a
research terminal rather than a marketing page.

Three reference aesthetics, blended:
  - Bloomberg Terminal: dense panes, function-key references, status bars,
    amber/cyan-on-black, ticker rhythms, no decoration.
  - Palantir Gotham/Foundry: classified-document gravity, low-chroma greys
    with one sharp accent, monospace + Inter mix, line-art icons.
  - Unix terminal: prompt, blinking cursor, ASCII headers, file-tree nav,
    keyboard-first ergonomics.

Three non-negotiable principles:
  1. KEYBOARD-FIRST. The site must be fully usable from the keyboard. A
     command palette (Cmd+K) is the canonical navigation. Mouse is a
     fallback, not a primary.
  2. DENSITY OVER WHITESPACE. Show information. Trust the visitor to read.
     The site is for readers, not skimmers. (But typography must remain
     legible — density does not mean cramped.)
  3. PER-PROJECT BESPOKE. There are very few projects (3 confirmed:
     Hivemind, Purposeless Efficiency, Theseus). Each gets its own custom
     dossier page. The system supports a generic project template, but the
     three featured projects override it heavily.

The "weird/never-seen" navigation gimmick: the landing page is a horizontal
accordion of vertical panels — one per top-level destination (About,
Hivemind, Purposeless Efficiency, Theseus, Resume). Hover/click expands a
panel into a full workspace; the others compress to monospace spines that
remain visible as a status rail. Cmd+K bypasses the accordion entirely.
